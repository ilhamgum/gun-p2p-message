# React using GUN for P2P Chatting

## How to use
This app is using React, Express and Gun as library to run

### Preparing
in `gun-p2p-message/` run
```
yarn install
```
then, 
```
cd ./client/
yarn install
```

### Run the apps
assume you're in the rootdir
```
yarn test
```
in new terminal:
```
cd ./client/
yarn start
```
new tab will open in browser with `http://localhost:3000`

Enjoyy..
